 site is live at https://gaurav-wb.github.io/HTML-CSS-Calculator-UI-Only-/

A modern calculator interface built using **pure HTML and CSS**, focused entirely on **UI/UX design and animation**.

##  Features
- Smooth **entry animations** (buttons slide from different directions)
- Clean **glassmorphism design**
- Responsive grid layout
- Interactive **hover and click effects**
- Minimal and professional UI

## Limitations
This project is **UI-only**. No JavaScript is used, so calculations are not functional.

## Purpose
Designed to demonstrate strong front-end fundamentals in:
- Layout structuring (CSS Grid)
- Advanced animations (keyframes + delays)
- Visual design principles
